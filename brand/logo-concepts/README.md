## 仓库名建议

最推荐：

```text
duanyu-android
```

理由：

- 简短直接。
- 明确这是 Android App。
- 和产品名 DuanYu 对齐。
- 后续如果有服务端、文档站、SDK，可以继续扩展：
  - `duanyu-server`
  - `duanyu-docs`
  - `duanyu-sdk`
